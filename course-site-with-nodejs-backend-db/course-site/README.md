# Course Site Frontend (React)

This is the React frontend for the Course Site app.

## Features
- Modern, responsive UI for course and lesson management
- Connects to the Node.js/Express backend API

## Getting Started
1. Install dependencies: `npm install`
2. Start the development server: `npm start`
3. The app runs on [http://localhost:3000](http://localhost:3000)

## Project Structure
- `src/` — React components and logic
- `public/` — Static assets

## Environment
- Configure API endpoints in `.env` if needed
